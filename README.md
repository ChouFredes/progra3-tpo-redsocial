**Trabajo Práctico – Programación 3
Red Social con Algoritmos de Grafos**
----------------------------------------------------
Este proyecto simula una pequeña red social utilizando Spring Boot y Neo4j.
Cada persona es un nodo y las relaciones de amistad son las aristas del grafo.
Sobre esta base se implementan distintos algoritmos vistos en clase
(BFS, DFS, Dijkstra, Kruskal, etc.) aplicados a un contexto de red social.

----------------------------------------------------
Tecnologías principales
----------------------------------------------------
- Java 17
- Spring Boot
- Neo4j 5
- Docker y Docker Compose
- Maven

----------------------------------------------------
Cómo ejecutar el proyecto
----------------------------------------------------
1. Clonar el repositorio:
   git clone https://github.com/ChouFredes/redsocial.git
   cd redsocial

2. Levantar los contenedores:
   docker compose up -d --build

3. Acceder desde el navegador:
   - Aplicación: http://localhost:8080
   - Neo4j Browser: http://localhost:7474
   - Usuario: neo4j
   - Contraseña: 12345678
